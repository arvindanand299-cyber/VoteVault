# VoteVault

A secure online voting system built using **Python (Flask)** and **Face Recognition** for biometric voter authentication. VoteVault ensures that only verified users can participate in elections, providing a secure and transparent voting experience.

---

## Features

- Face-based user registration
- Biometric login using facial recognition
- Secure voter authentication
- Online voting system
- Election dashboard
- Session-based authentication
- Responsive user interface

---

## Tech Stack

**Frontend**

**Backend**
- Python

- OpenCV

- NumPy

## Installation

1. Clone the repository

```bash
git clone https://github.com/arvindanand299-cyber/VoteVault.git
```

2. Navigate to the project folder

```bash
cd VoteVault
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python server.py
```
